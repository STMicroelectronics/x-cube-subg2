Release Notes for Projects of X-CUBE-SUBG2 Package

Copyright © 2021 STMicroelectronics

Purpose

The X-CUBE-SUBG2 Projects provide examples and applications to aid understanding how to develop an application using simple Point to Point communication, examples taken from S2-LP DK or more complex 6LoWPAN based connectivity, using Contiki-NG Middleware. Projects include STM32CubeMX project file (.ioc) for the graphical visualization of the STM32 microcontroller pins, peripherals and middlewares configuration.
